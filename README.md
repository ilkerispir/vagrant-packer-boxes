# Vagrant Box Packer Builds

## Requirements

### Build tool

* https://www.packer.io/
### VM Providers

* https://www.virtualbox.org/
* https://www.parallels.com/ 

## Packer Build Command
### x86-64 based CPU
```bash
packer build ubuntu-20.04-amd64.json
```

### ARM based CPU
```bash
packer build ubuntu-20.04-arm64.json
```
